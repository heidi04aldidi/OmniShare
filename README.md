# OmniShare

**OmniShare** is a specialized backend engine for circular economy asset sharing. It manages the complex lifecycle, safety requirements, and scheduling of community-owned resources.

## Project Documentation
This repository contains the architectural design for Milestone 1:

- [Project Vision & Scope](./docs/idea.md)

- [ER Diagram (Database Schema)](./docs/ErDiagram.md)
- [Class Diagram (OOP Architecture)](./docs/classDiagram.md)
- [Sequence Diagram (Logic Flow)](./docs/sequenceDiagram.md)

## Tech Stack
- **Backend:** NestJS (Node.js/TypeScript)
- **Database:** PostgreSQL with Prisma ORM
- **Patterns:** State Machine, Dependency Injection, Strategy Pattern.
